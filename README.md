### **mazda.atenza** ###
------

**mazda.atenza**是为了方便java开发，尤其是与webkit相关的开发，提升开发效率的一些列小工具的集成器，基于`swing`，界面完全手写，提供各种小功能，轻量，并保持更新

### **mazda.atenza**功能介绍  ###
------

> * 翻译：来自有道词典，自动识别语言
> * url编码：支持编码和转码
> * `unicode`编码：支持编码和转码
> * `json`格式化
> * 当前日期格式化
> * 未完待续
